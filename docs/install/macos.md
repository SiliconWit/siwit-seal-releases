# macOS

Tested on macOS 12 Monterey and later, both Apple Silicon and Intel.

!!! note
    macOS builds are coming soon. Track progress on the [Releases](https://github.com/SiliconWit/siwit-seal-releases/releases) page.

## Install (planned)

1. Download `SiliconWit-Seal-<version>.dmg` from Releases.
2. Open the `.dmg` and drag **SiliconWit Seal** into `Applications`.
3. The first launch may need: right-click → **Open** → confirm, since the build is not yet notarized.

## Uninstall

Drag **SiliconWit Seal** from `Applications` to the Trash. User config lives under `~/Library/Application Support/siwit-seal/`.
